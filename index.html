Cloud Balance – Final Year Project

Cloud Balance is a full-stack cloud cost and resource manager. It features secure IAM role-based AWS access, automatic resource & cost fetching, PostgreSQL storage, and a Kotlin-based Android app.

Architecture Diagram

GitHub Repositories

🔧 Backend API (Node.js + PostgreSQL) 📱 Android App (Kotlin + Jetpack Compose) 🚀 AWS Lambda Fetch Function (Cost & EC2) 📦 Terraform for Lambda Fetch 🛠️ Terraform for Backend Infrastructure

Key Features

  • 🔐 IAM Role input with external ID support
  • 📡 AWS STS-based cross-account role assumption
  • 💸 Fetches AWS cost and EC2 usage via Lambda
  • 🗄️ PostgreSQL schema includes users, roles, costs, resources, and alerts
  • 🧪 Tested with Mocha, CI via GitHub Actions
  • 📱 Kotlin app for interacting with backend securely
