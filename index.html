Cloud Balance – Final Year Project

Cloud Balance is a full-stack application that helps users monitor and manage their cloud resources and spending. It features AWS integration, resource tracking, cost analysis, alerting, and a mobile companion app.

Architecture Diagram

GitHub Repositories

🔧 Backend API (Node.js + PostgreSQL) 📱 Android App (Jetpack Compose) 🚀 AWS Lambda Fetch Function 📦 Terraform for Lambda Fetch Function 🛠️ Terraform for Backend Deployment

Key Features

  • 📡 Connect via IAM role to fetch AWS resources and cost data
  • 📊 Store and query data with PostgreSQL
  • 🛡️ Auth via Amazon Cognito (JWT)
  • ⚠️ Alert system for cost/resource events
  • 🧪 Tested with Mocha + GitHub Actions CI
